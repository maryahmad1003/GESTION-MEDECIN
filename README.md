# Application Gestion des Médecins - Réponses aux Questions

## Question 4 : Intérêt d'utiliser un modèle (interface) dans Angular

Un modèle définit la structure des données. Il permet de :
- Typage fort des données (TypeScript)
- Validation de la structure objet au temps de la compilation
- Auto-complétion dans l'IDE
- Documentation du code
- Réutilisation dans toute l'application

## Question 8 : Rôle d'un service dans l'architecture Angular

Un service est un provider de données et logique métier. Il permet de :
- Centraliser la gestion des données
- Partager les données entre composants
- Encapsuler la logique métier
- Rendre le code testable et réutilisable
- Respecter le principe DRY (Don't Repeat Yourself)

## Exécution

```bash
npm install
npm start
```

L'application sera disponible sur http://localhost:4200
